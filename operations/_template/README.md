# Operations Template

本目录是长周期 / 多阶段 / 连续推进任务的运行态文档模板。

生成命令：

```bash
node harness/core/operations/create-operation-docs.js <initiative>
```

默认生成到 `docs/operations/<initiative>/`。

生成文件：

- `current-<initiative>.md`
- `<initiative>-board.md`
- `<initiative>-matrix.md`
- `<initiative>-decisions.md`
